Popover directives/popover.js

Overview and examples

Fetches an external html partial and populates the popover with its content.

Use any button to trigger a popover menu by appending a bs-popover tag.

Both $scope.show() and $scope.hide() are available inside the popover to toggle its visibility.

popover: {{popover | json}}
Click to toggle popover
(won't close others) Unique Popover
(will close others) 
<a class="btn" bs-popover="'partials/popover/popover.html'">...</a>
<!-- force close every other popover with data-unique -->
<a class="btn" data-unique="1" bs-popover="'partials/popover/popover.html'">...</a>

Button directives/button.js

Overview and examples

Enables easy databinding between a button and an scope object.

Use any button with ng-model and bs-button tags.

button: {{button | json}}
Toggle me! 
<a data-toggle="button" class="btn" ng-model="button.active" bs-button">...</a>

Datepicker directives/datepicker.js

Overview and examples

Directive for bootstrap-datepicker (made by Stefan Petre).

Use any button to trigger a datepicker menu by appending a bs-datepicker tag.

You can use a custom button as toggler with data-toggle="datepicker" on a sibling element. Supports AngularJS validation out of the box.

datepicker: {{datepicker | json}}
<div class="control-group input-append">
  <input class="input-small" type="text" ng-model="datepicker.date" data-date-format="dd/mm/yyyy" bs-datepicker>
  <button class="btn" data-toggle="datepicker" type="button"><i class="icon-calendar"></i></button>
</div>
